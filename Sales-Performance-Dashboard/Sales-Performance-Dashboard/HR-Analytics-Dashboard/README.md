# HR Analytics Dashboard

## Project Overview
This Power BI dashboard analyzes employee data to provide insights into workforce trends, employee attrition, department performance, and workforce demographics. The dashboard helps organizations make informed HR decisions.

## Tools Used
- Power BI
- SQL
- Microsoft Excel
- Power Query
- DAX

## Key Insights
- Employee Attrition Rate
- Department-wise Headcount
- Gender Distribution
- Age Group Analysis
- Job Role Analysis
- Employee Performance Overview
- Hiring Trends
