# Sales Performance Dashboard

## Project Overview
This Power BI dashboard analyzes sales data to monitor business performance across products, regions, and customer segments. The dashboard provides interactive visualizations and KPIs to support data-driven decision-making.

## Tools Used
- Power BI
- SQL
- Microsoft Excel
- Power Query
- DAX

## Key Insights
- Total Sales
- Total Profit
- Sales by Region
- Sales by Category
- Monthly Sales Trend
- Top Performing Products
- Customer Segment Analysis
